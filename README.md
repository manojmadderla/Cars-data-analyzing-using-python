# 🚗 Car Data Analysis using Python

## 📌 Project Overview

This project focuses on analyzing automobile data using Python to uncover insights about different car manufacturers, vehicle specifications, pricing, and origins. The analysis is performed using data exploration and aggregation techniques to better understand trends in the automotive industry.

The project utilizes **Pandas**, **NumPy**, and **Matplotlib** for data manipulation and analysis.

---

## 🎯 Objectives

* Explore and understand the car dataset.
* Analyze car manufacturers and their vehicle counts.
* Identify vehicles with high weight specifications.
* Compare car prices across different regions of origin.
* Generate meaningful insights from automobile data.

---

## 🛠️ Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Jupyter Notebook

---

## 📂 Dataset Information

**Dataset:** `2. Cars Data1.csv`

The dataset contains information such as:

* Car Make (Manufacturer)
* Model
* Vehicle Type
* Origin
* MSRP (Price)
* Weight
* Engine Specifications
* Fuel Efficiency

## 📊 Analysis Performed

### 1️⃣ Data Exploration

* Display dataset records
* View first few rows using `head()`
* Inspect column names
* Check dataset structure and data types
* Generate descriptive statistics

### 2️⃣ Data Cleaning

* Check for missing values
* Identify duplicate records

### 3️⃣ Manufacturer Analysis

* List unique car manufacturers
* Count the number of vehicles produced by each manufacturer

### 4️⃣ Vehicle Weight Analysis

* Filter vehicles with weight greater than 4000 lbs
* Identify heavy-duty vehicles

### 5️⃣ Origin-Based Price Analysis

* Calculate total MSRP by origin
* Find highest-priced vehicles by origin
* Find lowest-priced vehicles by origin
* Compare automotive markets across regions

---

## 📈 Key Insights

* Understand which manufacturers dominate the dataset.
* Compare vehicle pricing trends among different origins.
* Identify premium and budget vehicle segments.
* Analyze distribution of heavy vehicles.
* Explore regional differences in automobile markets.
